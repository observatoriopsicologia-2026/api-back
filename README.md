# Observatorio - API Backend

Este repositorio contiene el backend del proyecto Observatorio, desarrollado con Node.js.

##  Estructura del proyecto

api-back/
 backend/
  src/
   package.json
   package-lock.json
   .env.example
   .gitignore
   README.md

## Instalación

Clonar el repositorio:

git clone https://github.com/observatoriopsicologia-2026/api-back.git
cd api-back/backend

Instalar las dependencias:

npm install

## Tecnologías utilizadas

* Node.js
* Express (o el framework que estés usando)
* Otras dependencias definidas en `package.json`

## Notas importantes

* El backend se encuentra dentro de la carpeta `/backend`
* No ejecutar `npm install` en la raíz del proyecto
* No subir archivos `.env` ni `node_modules`
